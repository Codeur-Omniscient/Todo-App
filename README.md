# Todo App

![Home image](./screely-1729551571645.png)

## Get Started

```bash
#First you need to clone this project
git clone https://github.com/Codeur-Omniscient/Todo-App

#Change directory
cd Todo-App

#Install dependencies
pnpm install
```

## Stack

- ReactJs
- Typescript
- TailwindCss
- Daisy Ui
- Framer Motion

## Feature

- Create Task
- Update Task
- Delete Task
- Search Task
- LocalStorage
- State Management
